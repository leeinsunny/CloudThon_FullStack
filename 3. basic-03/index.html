[ 목록나열 - OL,UL,DL ]

< OL태그 >

1. ol태그를 이용해서 정렬하기

  1. 딸기
  2. 바나나
  3. 그래놀라

2. ol은 reversed 속성을 통해서 내용 말고 앞에 있는 숫자만 역순으로 지정할 수 있다

  1. 첫째날인데
  2. 화이팅하자
  3. 근데 언제 끝나냐

3. ol은 start 속성을 통해서 시작하는 번호를 지정해줄 수 있다

  1. 딸기존맛
  2. 바나나대박
  3. 그래놀라..는 나름 먹어

4. ol은 type 속성을 통해서 알파벳으로도 올더링이 가능하다

  1. 그니까
  2. 오늘은 운동을
  3. 하체를 해야겠다

< UL태그 >

1. 일반 ul - 앞머리가 동그라미인 경우

  • 요거트 아이스크림이 아니라 그릭요거트를 시켰구나..ㅎㅎ
  • 근데 진짜 벌꿀집이 신의 한 수였음

2. 일반 ul - 앞머리가 직사각형인 경우

  • 그릭요거트 메이커
  • 사야겠다 진짜로

< DL태그 >

dl태그는 정의 목록으로, 백과사전처럼 항목이 나열되는 경우 사용

그릭요거트
- 이 시대 최고의 크림이라고 생각한다
- 벌꿀집과 그래놀라만 있다면 와플대학은 물리칠 수 있다
